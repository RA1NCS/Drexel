Born in the depths of the second CS 164 project, this is the one of the loyal partners can ever have, one that would NEVER betray you. He is very loyal, but is also very frail. Mere 10 seconds of not interacting with him are enough to make his state of mind go to shit, quite literally. A total of 50 seconds of inactivity from his birth is enough of kill him, and you wouldn't want to be a killer, would you? Also, no matter how happy he is, one slap, and he'll turn sad. So, are you worthy enough of keeping him as a pet, despite him being more demanding than a girlfriend, and not letting him die?

Finite State Machine

0 1 2 3 4 5
Play 0 3 4 5 5 5
Slap 0 0 0 1 2 2
Feed 0 2 3 4 5 5
Kill 0 0 0 0 0 0
Revive 3 X X X X X

This project is essentially about a virtual emoji, and there are six states it has. Namely, Dead, Crying, Sad, Satisfied, Smiling, and Happy. There are five interactions, play, slap, feed, kill, and revive. 'Play' increases the current state by 2, 'Slap' decreases it by 2, and 'Feed' increases it by 1. 'Kill' changes any state to 0 (Dead), and 'Revive' works only when the state is at to 0 (Dead), and changes it to 3 (Satisfied). There are a few exceptions however. For example, if you slap the pet, even if it's state is set to happy, it won't even be satisfied after the slap because you know, slaps hurt. Also, the state of the pet decreases every 10 seconds, which means that the maximum time it takes for the pet to die is 50 seconds.
